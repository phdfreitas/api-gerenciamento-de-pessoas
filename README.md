# Api Gerenciamento de Pessoas

# Objetivo
API básica para colocar em prática conhecimentos além de Java e Spring Boot

# Funcionalidades
- [x] Criar uma pessoa
- [x] Editar uma pessoa
- [x] Consultar uma pessoa
- [x] Listar todas as pessoas
